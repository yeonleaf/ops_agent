TV를 선택한 경우
상품기준명: 상품기준명을 입력한다
대상해상도: 대상해상도를 선택한다.
소장여부: 소장여부를 선택한다. PPV 상품스케쥴 등록 시 365일시청 여부도 체크하여 동일한 조건의 목록만 조회된다.
상품가격: 판매가격이 아닌 원가격을 입력한다.
판매가격: 상품팩별로 판매가격을 입력한다.
구매시청일: 구매시청을 입력함. 소장인 경우 9999일로 자동 고정되고 수정불가함.
과금기준: 영화인 경우 과금기준을 선택함.
할인구분: 특정한 할인이 지정된 경우 할인구분 코드를 선택함. 해당 코드 선택 시 특별한 제약 사항은 없음
VAT 구분: 판매가격이 VAT 포함가격인지를 지정함. 서비스에 노출 시 해당 정보에 따라 가격 계산이 달라지므로 유의 필요.
만료일 설정(시간): 지상파유효기간을 지정하는 정보로, 시간단위로 지정한다. 지정된 시간만큼 PPV 상품스케쥴 등록 시에 상품판매만료일을 [상품판매시작일시+만료일시간]로 설정한다.
