RAG 데이터 관리자 테스트 문서

이 문서는 RAG 데이터 관리자의 기능을 테스트하기 위한 샘플 문서입니다.

## 주요 기능

1. 문서 업로드
   - 다양한 형식의 문서 지원 (DOCX, PPTX, PDF, XLSX, TXT, MD)
   - 다중 파일 동시 업로드 가능

2. 문서 처리
   - FileProcessor를 통한 자동 문서 분석
   - 텍스트 추출 및 청크 분할
   - 메타데이터 생성

3. 벡터 데이터베이스 저장
   - 텍스트 임베딩 생성
   - ChromaDB에 저장
   - 검색 가능한 형태로 변환

## 사용 방법

1. 웹 UI에서 "RAG 데이터 관리자" 탭 선택
2. 파일 업로드 영역에 문서 드래그 앤 드롭
3. "업로드된 파일 처리 및 임베딩" 버튼 클릭
4. 처리 완료 후 벡터 DB에서 검색 가능

## 기술 스택

- Streamlit: 웹 UI 프레임워크
- LangChain: LLM 및 임베딩 처리
- ChromaDB: 벡터 데이터베이스
- Azure OpenAI: 임베딩 모델

이 문서가 성공적으로 처리되고 벡터 DB에 저장되면, RAG 시스템에서 검색 가능한 지식이 됩니다.
